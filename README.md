# Vue 3 Course

A comprehensive Vue 3 course demo project covering all major concepts from the official documentation.

## Features

- **Vue 3 Composition API** (`<script setup>`)
- **TypeScript**
- **Vue Router**
- **Pinia** State Management
- **i18n** (English / Chinese)
- **Dark Mode** support
- **Demos** for all core concepts

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
